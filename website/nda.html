This Non-Disclosure Agreement ("Agreement") is entered into between StreamAutoClipper ("Disclosing Party") and the alpha tester ("Receiving Party") who agrees to test the StreamAutoClipper alpha software.

2. Definition of Confidential Information

For purposes of this Agreement, "Confidential Information" includes ONLY the following specific items:

  • Alpha Access Codes: Your specific access code and any unused access codes
  • Software Files: The actual application files, executables, and installation packages
  • Screenshots/Recordings: Visual documentation of the software interface and functionality
  • Source Code: Any visible source code, technical implementation details, or reverse-engineered information
  • Competitive Use: Using the software to build competing products or services

IMPORTANT: You are encouraged to discuss your experience, opinions, and results publicly during streams and on social media!

The Receiving Party agrees to:
